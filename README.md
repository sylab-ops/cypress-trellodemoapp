# trellodemoapp
A simple trello clone application built on vue and json-server. The point of this app is to be super easy to install and run, so that you donâ€™t need to set up complicated database and have several scripts running in order to open app. 

The original version of this app was built by Zack Thoutt and updated Filip Hric, you can find it here https://github.com/filiphric/trelloapp

How to execute the project?
- npm install 
- npm start

That should do it. If you update the code, use npx grunt dev to build the app again.
